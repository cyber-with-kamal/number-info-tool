# 📱 Number Info Tool

A Python OSINT tool to fetch mobile number details.

## 🔥 Features
- Country detection
- Carrier info
- Phone validation
- OSINT links

## 🚀 Installation

git clone https://github.com/cyber-with-kamal/number-info-tool.git

cd number-info-tool

pip install -r requirements.txt

## ▶️ Usage

python3 number.py
